# Healthcare Appointment Booking App

A modern healthcare appointment booking mobile application prototype designed in Figma to simplify doctor discovery, appointment scheduling, and patient healthcare management.

## Overview

The Healthcare Appointment Booking App focuses on delivering a seamless digital healthcare experience for patients by enabling:

* Easy doctor discovery
* Appointment scheduling
* Time slot booking
* Patient profile management
* Secure and intuitive healthcare workflows
* Modern mobile-first UI/UX

This project was designed as a healthcare technology solution prototype with a strong focus on usability, accessibility, and clean user experience.
